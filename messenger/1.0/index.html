Messenger 1.0

  • Vendor: Messenger
  • Version: 1.0
  • Release Date: June 2, 2016
  • Emojis: 1181

New Facebook Messenger emojis were released on Android and iOS on June 2, 2016.

Support for skin tone modifiers is new in this release, and police officer, runner, pedestrian, surfer, and swimmer display as women.

About Messenger Emojis

Facebook has a unique emoji set that is displayed within Messenger for iOS, Android and web. These emojis are different to those used on the Facebook Desktop Website.
