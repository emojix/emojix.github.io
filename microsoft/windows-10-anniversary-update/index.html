Microsoft Windows 10 Anniversary Update

  • Vendor: Microsoft
  • Version: Windows 10 Anniversary Update
  • Release Date: Aug. 2, 2016
  • Emojis: 1648

Windows 10 Anniversary Update (version 1607), aka "Emojiversary Update" was released on August 2, 2016 and included a complete redesign of every emoji in the Segoe UI Emoji font. This release also added support for Unicode 9 emojis, as well as 52,000 family combinations including single parent families and interracial families. More about this update.

About Microsoft Emojis

Emojis from Microsoft are supported on PCs or tablets running Windows or phones running Windows Phone. This includes all brands of Windows PC, Microsoft Surface, and Windows Phones such as the Lumia range.

Support for color emoji presentation on Windows is limited. Many applications on Windows display emojis with a black and white text presentation instead of their color version.
