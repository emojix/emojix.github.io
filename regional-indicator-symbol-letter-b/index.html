🇧 Regional Indicator Symbol Letter B

The letter B, which can be used as part of a regional indicator pair to create emoji flags for various countries.

Regional Indicator Symbol Letter B was approved as part of Unicode 6.0 in 2010 and added to Emoji 2.0 in 2015.
